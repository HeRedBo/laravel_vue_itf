## 接口简介
系统后台登录接口

## 接口详情

### 请求地址
/admin/login

### 请求类型
POST

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
| username | string | 是 | 用户登录名 | - | admin |
| password | string |  是 | 用户登录密码 | - |  |
| remember | number | 否 | 是否记住密码 | 0 | 0 |
